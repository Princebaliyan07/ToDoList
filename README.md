# 📝 Dark Mode To-Do List

A simple and responsive To-Do List web application built using **HTML**, **CSS**, and **Vanilla JavaScript**. The app helps users manage daily tasks with a modern dark UI and automatically saves tasks using the browser's Local Storage.

## ✨ Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- 🗑️ Delete tasks
- 💾 Tasks are saved in Local Storage
- 🔄 Tasks persist after page refresh
- 🌙 Modern dark mode interface
- 📱 Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## 🚀 How to Run

1. Clone the repository
2. Open `index.html` in your browser
3. Start adding your tasks!
